üì¶ Key Projects

PolkaVM

Repository: https://github.com/paritytech/polkavm

Description: A general purpose user-level RISC-V based virtual machine designed for smart contracts and high-performance execution.

Key Features:

  • Fast execution competitive with WebAssembly VMs
  • Single-pass O(n) compilation
  • Low memory footprint (128KB baseline)
  • Fully deterministic execution
  • Cross-platform support
  • Built-in debugging and profiling tools

pallet-revive

Repository: https://github.com/paritytech/polkadot-sdk/tree/master/substrate/frame/revive

Description: An experimental runtime module that provides functionality for deploying and executing PolkaVM smart contracts.

Key Features:

  • Executes PolkaVM smart contracts
  • Supports Solidity (via revive) and Rust
  • Smart-contract code stored once via code_hash
  • Weight-based execution limits
  • Non-cascading call failures
  • Experimental unstable interface support

Westend Network

Type: Polkadot's primary permanent testnet

Purpose: Testing environment for PolkaVM and pallet-revive development

Key Features:

  • Primary testnet for PolkaVM development
  • pallet-revive deployed to AssetHub parachain
  • Free testnet tokens (WND) for development
  • REMIX instance for Solidity development
  • ink! v6 support and integration

Passet Hub

Type: Temporary smart contracts testnet

Purpose: Testing environment specifically for pallet-revive smart contracts

Key Features:

  • Second instance of Asset Hub system chain
  • Contains pallet-revive for PolkaVM smart contracts
  • Stable, community-managed environment
  • Regular updates from Westend Asset Hub
  • Will be decommissioned in Q4 2025
